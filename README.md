電動ガン用ヘッドアップディスプレイ **“Glass Unit”** の取り扱い説明です。

# 各設定方法

電源ON後、通常画面からエンターキー（⏎）を押すとメインメニューに切り替わります。  
メインメニューには以下の6項目があります。

- モデルセレクト（default, 1～3）
- モデルネーム編集
- HUD セッティング
- シューティングレンジキャリブレーション（実射キャリブレーション）
- データリセット
- HUDに戻る

この中で特に重要な  
**シューティングレンジキャリブレーション（実射キャリブレーション）** について解説します。

簡単なイメージとしては、10m～50mまでの距離で、さまざまな姿勢で銃を撃ち、その着弾点をユニットに記憶させる「マルチゼロイン」です。

各レンジ（10m～50m）、各姿勢  
（左傾90度 / 左傾45度 / 0度 / 右傾45度 / 右傾90度）  
でセンターマークを基準に照準し、着弾点を赤い○でプロットしていきます。

複数の着弾点を記憶させることで、銃身の上下左右の傾き、オフセット、弾速などをマイコンが計算し、HUDモード中はリアルタイムに着弾点を描画します。

まずは **20m～30m** あたりからキャリブレーションを始めるとよいでしょう。

コツとして、左傾45度をプロットした後は、必ず反対側の右傾45度もプロットしてください。  
片側のみでは、銃身の歪み・ホップ・重力影響をマイコンが正しく判断できません。  
必ずバランスよく対になるようキャリブレーションしてください。

# その他の基本的な取り扱い方

十字キーは **長押しで連打モード** に入ります。  
モデルネーム編集やキャリブレーション時は、チョン押しと長押しを併用してください。

モデル登録数は **3つ** です。  
それぞれ個別にモデルネーム、HUD設定、実射キャリブレーションを登録できます。  
複数の銃で相性を試してみてください。

### デモ動画

[![Demo Video](https://img.youtube.com/vi/BgiEXVQi1ko/hqdefault.jpg)](https://www.youtube.com/watch?v=BgiEXVQi1ko)

本製品は防水仕様（IPX4～5相当）です。  
フィールドプレイ中の大雨や泥水でも破損することはありませんが、プレイ後は電池ボックスを開けて十分に乾燥させてください。

レンズはプラスチック製のため、BB弾の直撃で損傷するおそれがあります。  
ゲーム中は **レンズガード兼遮光用ポリカーボネート板** を装着してください。

レンズ清掃は、水を含ませた柔らかい布またはティッシュで拭き、その後乾拭きしてください。  
薬品や洗浄剤の使用は避けてください。

# ゲーム中の操作方法

- 左右キー：距離レンジ（10m～50m）選択
- 上キー：LCOS（予測照準）ON / OFF
- 下キー：HUDモード / CQBモード切替

CQBモードでは、  
大きい順に **10m / 20m / 30m** のレクティルが表示されます。

レンジ選択の目安は、画面右側の縦長のレンジボックスです。  
設定レンジでの **身長170cmの人間サイズ** とほぼ一致します。  
対象物と比較しながら活用してください。

エンターキー（⏎）でメインメニューに戻ります。
