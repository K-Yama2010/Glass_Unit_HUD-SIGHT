# 各設定方法
電源ON後、通常画面からエンターキー（⏎）を押すとメインメニューに切り替わります。<br>
メインメニューには６つの項目があります。<br>
<br>
　・モデルセレクト（defoult,1～3)<br>
　  ・モデルネーム編集<br>
  　・HUD セッティング<br>
　・シューティングレンジキャリブレーション（実射キャリブレーション）<br>
  　・データリセット<br>
 　 ・HUDに戻る<br>
<br>
  この中で特に重要な **シューティングレンジキャリブレーション（実射キャリブレーション)** について解説します。<br>
  簡単なイメージとして、10ｍ～50ｍまでの距離で様々な姿勢で銃を撃ちその着弾点をユニットに記憶させる、マルチゼロインと捉えていただければいいかと思います。<br>
<br>
  各レンジ（10ｍ～50ｍ）、各姿勢（左傾90度　左傾45度　0度　右傾45度　右傾90度）でセンターマークで照準し、どこに着弾したかを赤い〇でプロットしていきます。<br>
  複数の着弾点の位置を記憶させることで銃身の上下左右の傾きやオフセット、弾速等までをマイコンが計算し、ＨＵＤモード中はリアルタイムにて着弾点を描画します。<br>
  <br>
  <br>
  まずは20ｍ～30あたりからキャリブレーションを始めるとよいでしょう。<br>
  コツとしては左傾45度の着弾点をプロットしたあとは反対側の右傾45度もプロットすることです。<br>
  片方だけのキャリブレーションでは銃身の曲がりなのか、ホップや重力の影響なのかをマイコンが判断できません。バランスよく対になるようキャリブレーションしていってください。<br>

# その他の基本的な取り扱い方
<br>

 <br>
  十字キーは長押しで連打モードに入ります。モデルネームで文字送りするときやキャリブレーションで着弾点をプロットするときはチョン押しと長押しを併用してください。<br>
  <br>
    モデル登録数は３つです。それぞれ個別にモデルネームやHUD設定、実射キャリブレーションを登録できますのでいろんな銃で相性を確かめてみてください。<br>
  <br>
   [![Demo Video](https://img.youtube.com/vi/BgiEXVQi1ko/0.jpg)](https://www.youtube.com/watch?v=BgiEXVQi1ko)　<br>
   <br>
  本製品は防水仕様（IPX4～5相当）で、フィールドプレイ中の大雨や泥水に対しても壊れたりすることはありませんが、プレイ後は電池ボックスを開ける等乾燥を心がけてください。<br>
  <br>
  レンズはプラスチック製でBB弾等の直撃を受けると損傷するおそれがあります。ゲーム中はレンズガード兼遮光のポリカーボネイト板を装着してください。<br>
  <br>
  また、レンズの手入れには水を含んだ柔らかい布かティッシュで行い、その後に乾拭きしてください。薬品や洗浄剤の使用はお控えください。<br>
  <br>

# ゲーム中の操作の仕方
<br>
十字キーの左右キーで対象距離レンジ10ｍ～50ｍまでの選択、上キーでLCOS（予測照準）のON/OFF,下キーで通常のHUDモードとCQBモードの切り替えを行います。CQBモードで表示されるレクティルは大きい順から10ｍ、20ｍ、30ｍです。<br>
<br>
レンジ選択のコツは画面右側の縦長の長方形（レンジボックス）のサイズです。設定レンジでの身長170㎝の人間の大きさとほぼ同等のサイズです。対象物と対比しながら有効に活用してください。<br>
 <br>
 エンターキー（⏎）を押すとメインメニュー（設定項目）に切り替わります。<br>
 <br>
